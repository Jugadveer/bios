# WealthPlay Financial Literacy Courses

This document lists all courses available in the WealthPlay platform and outlines the content requirements for each course to create comprehensive, high-quality learning materials.

## Course Structure

Each course follows this structure:
- **Course ID**: Unique identifier (used in URLs and data files)
- **Title**: Short, crisp course name
- **Overview**: Brief description of what the course covers
- **Duration**: Estimated learning time
- **Modules**: Individual learning units within the course
  - Each module has:
    - **ID**: Unique module identifier
    - **Title**: Module name
    - **Summary**: Brief overview of module content
    - **Fixed Q&A**: Pre-defined question-answer pairs for quick responses

## Content Development Guidelines

### For Each Course Folder:

1. **Create detailed content files**:
   - `content.md` - Main course content (comprehensive explanations)
   - `examples.md` - Real-world examples and case studies
   - `exercises.md` - Practical exercises and worksheets
   - `visuals.md` - Descriptions of needed visual content
   - `glossary.md` - Key terms and definitions
   - `resources.md` - Additional learning resources

2. **Content Format**:
   - Use clear headings and subheadings
   - Include practical examples relevant to Indian context
   - Use simple language suitable for beginners
   - Include visual descriptions where charts/graphs would help
   - Add real-world scenarios and case studies

3. **AI Integration**:
   - The AI mentor will use content from these folders to provide context-aware answers
   - Content will be processed for RAG (Retrieval Augmented Generation)
   - Ensure content is structured for easy chunking and embedding

4. **Quality Standards**:
   - Accuracy: Verify all financial information
   - Relevance: Focus on actionable advice
   - Clarity: Use simple language, avoid jargon
   - Completeness: Cover all key concepts
   - Practical: Include actionable steps and tools
